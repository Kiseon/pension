# New User Onboarding

신규 사용자의 목표는 가능한 적은 마찰로 첫 번째 신뢰 가능한 노후 수입
프로젝션을 얻는 것이다.

## Onboarding principles

- 사용자가 모르는 값은 추정값으로 입력할 수 있게 하되, 결과에서 신뢰도 차이를 표시한다.
- 입력 단계마다 왜 필요한 정보인지 설명한다.
- 배우자 정보는 선택 가능하지만, 부부 가구의 프로젝션 품질에는 큰 영향을 준다는 점을 안내한다.
- 민감한 자산 정보는 저장 여부와 삭제 방법을 명확히 고지한다.

## Flow

### 1. Household setup

입력 항목:

- 본인 생년월일
- 배우자 생년월일 또는 배우자 없음
- 분석 시작 월
- 현재 재직 여부
- 현재 월 세후 근로소득
- 은퇴 목표 연령
- 목표 은퇴 후 월 세후 수입

검증:

- 생년월일이 미래가 아닌지 확인한다.
- 분석 종료 월은 본인이 100세가 되는 해의 12월로 고정한다.
- 배우자 생년월일이 입력된 경우 두 사람의 월별 연령을 독립적으로 계산한다.
- 한국 원화 기준으로 입력받으며 국가/통화 선택은 제공하지 않는다.

### 2. Employment and retirement events

입력 항목:

- 현재 직장 상태와 예상 퇴직 월
- 은퇴 목표 연령
- 월 세후 근로소득
- 퇴직금 예상액 또는 산정에 필요한 평균임금/근속기간
- 희망퇴직 가능성, 위로금 산식 또는 예상 금액
- 퇴직 이후 목표 월 세후 수입

결과 활용:

- 은퇴 목표 전후의 월별 수입 변화를 연결해서 보여준다.
- 갑작스러운 퇴직 또는 희망퇴직 시나리오에서 일시금이 목표 월수입을 얼마나 보완하는지 계산한다.
- 목표 월수입 대비 부족액과 은퇴 전 추가 저축 필요액을 산출한다.

### 3. Real estate income and costs

입력 항목:

- 자산명
- 현재 추정 가치
- 보유 지분율
- 월세 또는 임대료
- 월별 비용: 관리비, 유지보수비, 재산세, 대출 원리금, 기타 비용
- 연간 수입 상승률
- 주택담보 연금 검토 여부

검증:

- 자산 유형은 필수 구분하지 않는다.
- 임대/사업 수입과 비용을 분리해서 받는다.
- 공동 소유는 기본 50:50으로 나눈다.
- 매각 및 공실률은 초기 범위에서 제외한다.

### 4. Pension and recurring income

입력 항목:

- 국민연금
- 퇴직연금
- 개인연금
- 보험성 연금
- 주택담보 연금
- 연금별 목표 월 세후 수령액
- 희망 수급 개시 연령 또는 개시 월
- 연금별 물가연동 또는 상승률
- 기타 반복 수입

검증:

- 월 지급인지 연 지급인지 명확히 선택한다.
- 세후 금액 기준으로 저장한다.
- 시작 월과 종료 월이 충돌하지 않는지 확인한다.
- 국민연금, 주택담보 연금처럼 수급 개시 시점에 따라 금액이 달라지는 항목은 공식/공신력 있는 출처 기반 규칙 버전으로 자동 산정한다.

### 5. Savings, IRP, and investments

입력 항목:

- 현금 및 예금
- IRP 현재 잔액
- IRP 월/연 입금 계획
- 채권, 펀드, ETF, 주식 등 투자금
- 현재 주식 투자액
- 자산별 연간 예상 수익률
- 배당 또는 이자 수입
- 추천 인출 전략 계산에 사용할 제약조건
- 위험 수준

검증:

- 예상 수익률은 시나리오 가정으로 분리한다.
- 원금 인출과 수익 수입을 구분한다.
- 투자 손실 가능성은 보수 시나리오에서 반영한다.
- 목표 월수입 달성을 위해 필요한 인출액, 원금 소진 가능성, 추가 저축 필요액을 계산한다.

### 6. Assumptions review

사용자가 확인해야 할 기본 가정:

- 물가상승률
- 임대료 상승률
- 투자 수익률
- IRP 및 주식 예상 수익률
- 국민연금 조기/정상/연기 수령 규칙 버전
- 주택담보 연금 산정 규칙 또는 계산기 참조 버전
- 임대소득세, 재산세, 종합소득세 월별 비용화 규칙
- 연금 개시 규칙

### 7. First projection

결과 화면은 다음을 제공한다.

- 월별 총수입
- 수입원별 breakdown
- 본인과 배우자의 해당 월 연령
- 은퇴 후 목표 월 세후 수입 대비 충족률
- 수입 공백 구간
- 추천 연금 개시 시점 및 금융자산 인출 전략
- 은퇴 목표까지 필요한 추가 저축액
- 입력 신뢰도와 누락 정보
- 프로젝션 설명 및 다운로드

## Success metrics

- 첫 프로젝션 생성 완료율
- 입력 중간 저장 후 복귀율
- 필수 입력 대비 누락률
- 사용자가 기본 가정을 직접 검토한 비율
- 프로젝션 결과에서 설명 패널을 연 비율
- 사용자가 자산 정보를 갱신하는 빈도

## Edge cases

- 배우자가 없는 사용자
- 배우자 생년월일을 모르는 사용자
- 월세가 불규칙한 임대 자산
- 연금 시작 월이 이미 지난 사용자
- 목표 은퇴 전 희망퇴직이 발생한 사용자
- IRP 납입 계획이 중간에 변하는 사용자
- 주택담보 연금 개시 가능 연령 전후에 있는 사용자
- 부채 상환이 수입보다 큰 기간이 있는 사용자
