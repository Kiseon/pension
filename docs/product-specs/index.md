# Product Specs

이 디렉터리는 노후 수입 설계 서비스의 사용자 기능 명세를 관리한다.

## Specs

- [New User Onboarding](./new-user-onboarding.md)
- [Monthly Income Projection](./monthly-income-projection.md)

## Product surface

초기 제품은 다음 기능으로 구성한다.

1. 가구 프로필 입력
2. 재직 상태, 근로소득, 퇴직금, 희망퇴직위로금 입력
3. 부동산 수입/비용, 연금, IRP, 금융투자 자산 입력
4. 한국 제도 기반 연금 개시 시점별 추산
5. 재직 기간부터 100세 되는 해까지 월별 세후 수입 프로젝션
6. 목표 월수입 달성을 위한 인출/추가저축 추천
7. 입력 저장, 품질 점검, 설명 가능한 결과 리포트

## Spec template

각 제품 스펙은 다음 항목을 포함한다.

- 사용자 문제
- 주요 사용자
- 범위
- 입력 데이터
- 계산 또는 의사결정 규칙
- 결과 화면
- 신뢰성 요구사항
- 제외 범위
- 미해결 질문
