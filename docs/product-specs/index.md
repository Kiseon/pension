# Product Specs

이 디렉터리는 노후 수입 설계 서비스의 사용자 기능 명세를 관리한다.

## Specs

- [New User Onboarding](./new-user-onboarding.md)
- [Monthly Income Projection](./monthly-income-projection.md)

## Product surface

초기 제품은 다음 기능으로 구성한다.

1. 가구 프로필 입력
2. 자산 및 수입원 입력
3. 가정값과 시나리오 설정
4. 월별 노후 수입 프로젝션
5. 설명 가능한 결과 리포트
6. 입력 품질 및 신뢰도 점검

## Spec template

각 제품 스펙은 다음 항목을 포함한다.

- 사용자 문제
- 주요 사용자
- 범위
- 입력 데이터
- 계산 또는 의사결정 규칙
- 결과 화면
- 신뢰성 요구사항
- 제외 범위
- 미해결 질문
